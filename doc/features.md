# 项目功能列表 (AIGC_Helper)

该项目（AIGC_Helper）是一个基于 Java 和 Spring Boot 框架开发的 AIGC（人工智能生成内容）后台辅助服务。
通过对当前项目的代码解析，该项目主要包含以下核心功能模块：

## 1. 用户管理模块 (UserController)
*   **用户注册** (`/register`): 允许新用户创建账号。
*   **用户登录** (`/login`): 提供用户凭证验证，并生成和返回 JWT (JSON Web Token) 用于后续的身份认证。

## 2. 聊天交互模块 (ChatController)
*   **智能标题生成** (`/generate-title`): 核心的大模型/AI交互功能，利用 AIGC 能力根据上下文或是请求内容生成对应的聊天/内容标题。

## 3. 聊天会话与历史管理模块 (ChatHistoryController)
*   **查询用户会话列表** (`/sessions/{username}`): 获取指定用户的所有历史聊天会话 (Sessions)。
*   **创建新会话** (`/sessions`): 初始化并保存一个新的聊天会话。
*   **获取会话聊天记录** (`/messages/{chatId}`): 根据指定的会话 ID 获取具体的聊天消息历史。
*   **删除会话记录** (`/sessions/{chatId}`): 根据指定会话 ID 删除不需要的历史聊天会话。

## 4. 管理员功能模块 (AdminController)
*   **管理员注册** (`/register`): 针对系统后台管理员的独立注册功能。
*   **管理员登录** (`/login`): 针对系统后台管理员的独立登录认证与授权访问功能。

## 5. 游戏功能模块 (GameController)
*   **游戏相关操作**: 包含一些娱乐、游戏与交互功能（例如与AIGC相关的文字游戏或是趣味功能扩展）。

## 6. 系统安全与工具
*   **JWT 鉴权机制** (`JwtUtil.java`): 使用 `io.jsonwebtoken` 提供全系统的令牌生成、解析和鉴权机制，有效期设置为12小时，保障数据安全与接口调用的安全性。
*   **请求拦截与权限校验**: 支持基于 JWT 的令牌校验请求拦截以保护敏感接口。
